# 🎓 Smart Event Campus

Smart Event Campus adalah aplikasi berbasis **PHP Native** dan **MySQL** yang digunakan untuk mengelola informasi kegiatan kampus seperti Seminar, Workshop, Lomba, dan Pelatihan.

---

## 📌 Fitur

### 👨‍💼 Admin

- Login Admin
- Dashboard Admin
- CRUD Data Event
- Search Event
- Statistik Event
- Logout

### 👨‍🎓 Mahasiswa

- Login Mahasiswa
- Dashboard Mahasiswa
- Cari Event
- Detail Event
- Logout

---

## 💾 Cara Install

1. Jalankan XAMPP.
2. Aktifkan Apache dan MySQL.
3. Import database `smart_event.sql`.
4. Simpan project ke folder `htdocs`.
5. Buka browser.

```
http://localhost/smart-event-campus
```
